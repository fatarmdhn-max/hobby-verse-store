# HobbyVerse Store – Website Toko Hobi & Koleksi

## Deskripsi Project

**HobbyVerse Store** adalah sebuah website statis yang dirancang sebagai platform e-commerce untuk toko hobi dan barang koleksi (action figure, model kit, merchandise, dll). Project ini dikembangkan untuk memenuhi tugas mata kuliah **Pemrograman Web 1**.

Website ini menyajikan antarmuka yang modern, responsif, dan interaktif, memungkinkan pengguna untuk melihat katalog produk, informasi promo, galeri, serta menghubungi pihak toko melalui formulir kontak.

## Fitur Website

Beberapa fitur unggulan yang diimplementasikan dalam website ini meliputi:

- **Desain Responsif**: Tampilan website menyesuaikan dengan berbagai ukuran layar (desktop, tablet, mobile) menggunakan framework Bootstrap 5.
- **Dark Mode**: Fitur untuk mengubah tema tampilan antara mode terang (light) dan gelap (dark) yang disimpan dalam local storage browser.
- **Validasi Form Kontak**: Validasi input pada formulir kontak menggunakan JavaScript (memastikan email valid dan field tidak kosong).
- **Filter Kategori Produk**: Fitur interaktif untuk menyaring produk berdasarkan kategori tanpa reload halaman.
- **FAQ Interaktif**: Menggunakan komponen accordion untuk menampilkan pertanyaan dan jawaban secara rapi.
- **Galeri & Katalog**: Penataan layout grid yang rapi untuk menampilkan gambar produk dan koleksi.

## Teknologi yang Digunakan

Project ini dibangun menggunakan teknologi web standar:

- **HTML5**: Struktur konten halaman web.
- **CSS3**: Styling dan tata letak kustom.
- **Bootstrap 5**: Framework CSS untuk desain responsif dan komponen UI.
- **JavaScript (Vanilla JS)**: Logika interaktivitas (DOM manipulation, event handling).

## Catatan Tambahan

- Website ini bersifat **statis** (Front-End only).
- Tidak menggunakan database atau pemrosesan Back-End (PHP/Node.js).
- Seluruh data produk dan konten ditulis langsung dalam kode HTML (hardcoded).
- Dibuat semata-mata untuk keperluan akademik dan pembelajaran dasar pengembangan web.
